# 2D製図ツール(seizu-tool) 要件定義・仕様書 v1.0

> v0.1〜0.2 は壁打ちで合意した要件ドラフト。本 v1.0(2026-07-03)は**実装完了後に実情へ合わせて全面改訂した確定仕様**。
> 実装との対応: ソースは `src/`、テストは `test/`(node --test、111件)、配布物は `dist/seizu.html`。
> リポジトリ: https://github.com/oltotlo79-rgb/CAD

---

## 1. 目的・背景

- 工作物を製作する前の三面図作成にかかる工数を削減する。
- 3D CADで立体図を描いてから三面図を起こすワークフローを、Excel感覚で手早く線を引ける軽量な2D製図ツールで置き換える。
- 対象は主に機械部品の三面図(正面図・平面図・側面図)。

---

## 2. 対象環境・配布 【実装済み】

- **単一HTMLファイル**(`dist/seizu.html`、約70KB)として動作。外部依存・ネット接続不要。
- ブラウザで開くだけで動く。**Chrome / Edge 推奨**(上書き保存が使える)。他ブラウザでも保存はダウンロード方式で動作。
- 配布はファイル1つのコピー、またはGitHubからのダウンロード。
- 開発時のみ Node.js + esbuild を使用(`npm run build` で単一HTMLを生成)。
- 将来ファイル関連付け等のOS連携が欲しくなった場合は、同じコードをTauriで包んでデスクトップアプリ化できる(作り直し不要)。

---

## 3. アーキテクチャ 【実装済み】

- 描画は HTML Canvas。図形は「データオブジェクト」として保持し、状態変化のたびに再描画する。
- **すべての座標・寸法は実寸mmで保持**。縮尺と画面ズームは描画変換にのみ適用し、データ自体には掛けない。
- **線の太さ・破線パターン・文字高さ・矢印サイズは用紙上mm基準**(縮尺に依存せず、印刷時に常に同じ見た目)。
- 純粋ロジック(座標変換・幾何・寸法レイアウト・SVG生成など)はUIから分離したESモジュールで、Node.jsの組み込みテストランナーで検証している。
  - `papers` 用紙 / `viewTransform` 座標変換 / `gridCalc` グリッド / `geometry` 幾何 / `model` データモデル /
    `serializer` 保存形式 / `history` Undo / `snap` 点スナップ / `guides` 投影ガイド / `editOps` トリム・フィレット等 /
    `dims` 寸法・記号 / `hatch` ハッチング / `bom` 部品表 / `titleBlock` 表題欄 / `svgExport` SVG出力 /
    `snapshotStore` クラッシュ復元 / `renderer`・`app` UI層
- デバッグ・自動検証用に内部状態を `window.__seizu` で参照できる。

## 4. 単位・座標系・縮尺 【実装済み】

- 単位は mm 固定。角度は度(°)、反時計回り正。
- 内部座標は用紙左下を原点とする絶対mm・y軸上向き。実寸空間の原点は用紙原点に一致し、縮尺は用紙原点を中心に適用される。
- **ユーザー原点**を任意位置に設定できる(原点設定ツール)。座標の入力・表示はユーザー原点からの相対値。図形データは絶対座標で保持するため、原点を移動しても図形は動かない。初期原点は図面枠の内側・左下。
- 縮尺は図面全体で1つ。「1:5」「2:1」形式(全角コロン可)で数値入力。後から変更しても図面データは不変で、表示と寸法値は実寸のまま。
- 入力は実寸で行う。寸法値も実寸で自動表示される。

## 5. グリッドとスナップ 【実装済み】

### グリッド
- 実寸mm基準。**10 / 5 / 2 / 1 / 0.5 / 0.1 mm** の6段階(5・2mmはズーム自動切替を滑らかにするための中間段階)。
- 自動モード: 画面上の間隔が8px以上になる最小の段階を選ぶ。ズームアウトしすぎたら非表示。
- 手動固定モードあり。10mmの倍数の線を濃く表示。

### スナップ(それぞれON/OFF可)
- **グリッドスナップ**(「スナップ」チェック)
- **点スナップ**(「点スナップ」チェック): 端点・中点・円/円弧/楕円の中心・四半点・**要素同士の交点**(線分×線分、線分×円)。種類別のマーカー(□端点・△中点・○中心/四半点・×交点)でハイライト。
- **投影ガイド・45°線へのスナップ**: ガイド上の垂線の足・ガイド同士の交点・45°線との交点(§8)。
- 優先順位: 点スナップ/ガイド候補のうち最も近いもの → なければグリッド。
- **中心線モード**(線種=中心線で作図中): カーソル近傍の線分中点・円/楕円中心にマーカーと十字軸を表示し、中点そのもの、または**中点を通る水平/垂直軸上の任意の点**(自由座標はグリッド)に吸着する。軸上なら図形の外側でもよいため、外形線からはみ出した中心線が引ける。このモード中は端点・交点への通常スナップを無効化し、角に吸われるのを防ぐ。

## 6. 作図要素 【実装済み】

### 形状
直線、連続線(ポリライン)、矩形(回転可)、円(中心+半径点)、円弧(中心+始点+終角)、楕円(中心+コーナー、回転可)、**楕円弧**(中心→コーナー→開始→終了の4クリック、回転可)、スプライン(Catmull-Rom、制御点を通る)、文字(回転可)

### 線種(JIS準拠、太さ・破線は用紙mm)
| 線種UI | 用途 | 太さ | パターン | 配置レイヤー |
|---|---|---|---|---|
| 外形線 | 太実線 | 0.5mm | 実線 | outline |
| かくれ線 | 破線 | 0.35mm | 3-1.5 | hidden |
| 中心線 | 一点鎖線 | 0.25mm | 8-1.5-1.5-1.5 | center |
| 想像線 | 二点鎖線 | 0.25mm | 8-1.5-1.5-1.5-1.5-1.5 | outline |
| 補助線 | 細実線 | 0.25mm | 実線 | aux(印刷されない) |

### 注記・寸法・記号
文字、寸法一式(§9)、引出線、バルーン(部品番号・自動採番)、表面粗さ記号(Ra値)、幾何公差の公差記入枠。
**溶接記号は専用実装なし**(引出線+テキストで記入する運用)。

### 断面・組立
ハッチング(閉図形=矩形/円/楕円/閉じた連続線を指定。角度・間隔を入力可、既定45°/3mm)、部品表(BOM: 品番/品名/数量/材質。バルーンから行を自動生成、セルはダブルクリック編集)。

### 図面管理
図面枠(自動)、表題欄(§11)。

## 7. 作図・編集・視点操作 【実装済み】

### 作図
- クリック式(ドラッグ描画は選択の範囲指定に使用)。連続線・スプラインはEnterかダブルクリックで確定、Escで中止。
- **数値入力作図**: 始点X/Y(原点相対)・長さ・角度をパネルに入力して「作図」。マウス作図中は長さ・角度が実況表示され、値を入れてEnterで確定できる。

### 数値編集(選択して後から変更)
- 基本図形を**1つ選択するとパネルに現在値が入り、ボタンが「更新」に変わる**。ラベルは種別に応じて切り替わり、使わない欄は無効化される。
- 反映方法: **どの欄でもEnter / 欄からフォーカスを外す / 「更新」ボタン** のいずれでも即反映。
- パネルは選択種別ごとの動的フィールド(欄数可変):
  - **直線** = 始点X/Y + 長さ + 角度(始点基準: 長さ・角度の変更で終点が動く)
  - **円** = 中心X/Y + **直径φ**(機械図面の慣例に合わせ直径入力)
  - **円弧** = 中心X/Y + 半径R + 開始角 + 終了角
  - **矩形** = 左下X/Y + 幅 + 高さ + **回転**(左下角を中心に回転)
  - **楕円 / 楕円弧** = 中心X/Y + 半径X + 半径Y + **回転**(楕円弧はさらに開始角 + 終了角)
  - **文字** = 位置X/Y + **回転**
  - **連続線 / スプライン(全体)** = 始点X/Y(指定位置へ形状維持のまま平行移動)
- **線分単位の編集**: 選択中の連続線/スプラインをもう一度クリックすると最寄りの線分(スプラインは制御点間)が選択され(オレンジ強調+端点マーカー)、その線分の 始点X/Y + 長さ + 角度 を始点基準で個別編集できる。頂点の移動は隣接線分にも連続的に反映される。

### 編集
- 選択(クリック / 空白からのドラッグで範囲選択 / Shiftで追加除外)、ドラッグ移動、複製(Ctrl+D、+10mm)、削除(Delete)。
- **コピー&ペースト**: Ctrl+Cで選択図形を内部クリップボードへ、Ctrl+Vで**マウス位置(バウンディング中心)に貼り付け**(マウスがキャンバス外なら+10mmずらし)。何度でも貼り付け可。
- **右ドラッグ複製**: 図形を右ボタンで掴んでドラッグし、離した位置に複製を作る(移動先は破線枠でプレビュー、グリッド/点スナップ有効)。
- 回転90°、左右反転、上下反転(選択集合のバウンディング中心基準)。
- **トリム**(交点間を削除)・**延長**(最寄り要素まで)。対象は直線のみ。
- **オフセット**(距離指定の平行複製: 直線/円/円弧/矩形)。2段階操作: 対象をクリック→**ずらす側をクリック**(方向自由)。
- **フィレット**(半径指定で角をRに)・**面取り**(サイズ指定で角を切り落とし)。対象は直線2本を順にクリック、接点まで自動トリム/延長。
- **分解**: 選択した矩形・連続線を個別の直線に変換する(トリム/フィレット/面取りを矩形の角などに使う前処理)。
- 編集ツールが対象外の図形や条件不足で何もできない場合は、**ステータスバーに理由と次の操作をガイド表示**する。
- **Undo / Redo**(Ctrl+Z / Ctrl+Y、履歴100件、スナップショット方式)。
- ダブルクリック編集: 寸法値の上書き・文字・引出線・バルーン番号・粗さ値・公差枠・部品表セル・表題欄。

### 視点操作
- **スクロール=上下移動**、**Shift+スクロール / Ctrl+スクロール=拡大縮小**(カーソル基準。トラックパッドのピンチも可)。
- **縦・横スクロールバー**を画面右端・下端に常時表示。つまみのドラッグで移動、空き部分クリックでその位置へジャンプ。つまみの長さは表示範囲の割合を反映(スクロール範囲=用紙±半分+現在の表示範囲)。
- **＋/－ボタン**(画面中央基準)、**「全体」ボタン**(用紙全体にフィット)、**矢印キー**(60px相当)、中ボタンまたはSpace+ドラッグで自由パン。

## 8. 補助線(2種類) 【実装済み】

### 8a. 投影補助線(第三角法・一時ガイド)
- 「投影ガイド」ON時、**選択した要素**のスナップ点(端点・中点・中心・四半点)から水平/垂直の破線ガイドを用紙全体に表示する。一時ガイドであり保存・印刷されない。
- **45°ミラー線**: 「45°線配置」でクリック位置を通る傾き+1の線を置く(平面図↔側面図の奥行き転写用)。表示ON/OFF可。通過点は図面ファイルに保存される。
- ガイドの垂線の足・ガイド同士の交点・45°線との交点にスナップできる。

### 8b. 作図補助線(下書き線)
- 線種「補助線」で描いた要素は `aux` レイヤーに載り、**画面には見えるが印刷・SVG出力に含まれない**。下書き→なぞり運用に対応。

## 9. 寸法記入 【実装済み】

- 種類: **長さ**(水平・垂直は自動判定、Shiftで平行寸法)、**直径φ**(円をクリック)、**半径R**、**角度**(頂点+2点、3クリック目が円弧半径)、**面取りC**(面取り線をクリック+引出し位置)、**引出線**(対象→折れ点→文字入力)。
- 寸法値は実寸から**自動計測**(小数2桁まで、末尾ゼロなし)。**ダブルクリックで手動上書き**可(空にすると自動に戻る)。
- **公差はこの上書きで記入する**(例 `50±0.1`、`50 +0.1/0`)。専用の公差入力UIは持たない。
- 見た目: 寸法補助線(突き出し2mm)・寸法線・塗り矢印(長さ3mm)・文字高さ3.5mm(いずれも用紙mm)。垂直・平行寸法の文字は寸法線に沿って回転。
- 制約: 寸法は作成時の図形形状のコピーを持つ。**後から図形を変えても寸法は追従しない**(引き直すか値を上書きする)。

## 10. レイヤー 【実装済み】

- プリセット6レイヤー: 外形線 / かくれ線 / 中心線 / 寸法 / 注記 / 補助線。
- 「レイヤー」パネルでレイヤーごとに**表示ON/OFF・印刷ON/OFF**。補助線レイヤーは既定で印刷OFF。
- 要素は線種選択に応じて自動でレイヤーに割り当てられる(§6の表)。
- 任意レイヤーの追加は未対応(今後の候補)。

---

## 11. 図面枠・表題欄 【実装済み】

- 用紙サイズ: A2 / A3 / A4 / B3 / B4 / B5(JIS B列)。縦・横選択可。図面枠は全周10mm内側に自動描画。
- 表題欄は図面枠内側の**右下に固定配置**(幅90mm・行高8mm)。項目: 図番 / 図名 / 尺度 / 用紙 / 材質 / 作成者 / 日付。
- **尺度・用紙は図面設定から自動反映(bind)で手入力不可**。その他の欄はダブルクリックで記入。
- **日付は新規図面作成時に作成日(YYYY/MM/DD)で自動初期化**される(手動で書き換え可)。
- 項目の追加・削除・配置カスタムは未対応(データ形式上は `fields` 配列で拡張可能)。

---

## 12. ファイル形式・保存 【実装済み】

- ネイティブ保存: 独自JSON(§14)。続きから編集できる。
- **上書き保存+フォールバック**: Chrome/EdgeはFile System Access APIで開いたファイルへそのまま上書き(Ctrl+S)。非対応ブラウザはダウンロードに自動フォールバック。Ctrl+Shift+Sで名前を付けて保存。
- 開く: ダイアログ(Ctrl+O)または**JSONファイルのドラッグ&ドロップ**(D&DでもChromiumなら書込ハンドルを取得し上書き保存可)。
- **未保存のままタブを閉じようとしたら警告**。
- **クラッシュ復元**: 編集から2秒後にIndexedDBへ自動スナップショット(1世代)。次回起動時に未保存の変更があれば非モーダルのバナーで復元/破棄を選べる。ファイルへの自動保存ではない(正式な保存は手動のみ)。

---

## 13. 印刷・出力 【実装済み】

- **一次エクスポート形式: SVG**(用紙実寸mm、`width="420mm"` 等)。印刷OFFレイヤー・投影ガイド・グリッドは含まない。フォントは埋め込まず名前参照(Yu Gothic UI / Meiryo)。
- **PDF化はブラウザの印刷ダイアログ経由**(「印刷」ボタン→PDFに保存)。`@page` に用紙実寸を指定済み。**倍率100%(実際のサイズ)を指定すると実寸**になる。
- 1ファイル=1図面(1枚)。三面図・断面図・部品表はその1枚に自由配置する。

    { "id": "aux", "name": "補助線", "visible": true, "printable": false }
  ],
  "nextId": 12,
  "entities": [
    { "id": 1, "type": "line", "layer": "outline", "lineType": "solid", "x1": 0, "y1": 0, "x2": 100, "y2": 0 },
    { "id": 2, "type": "rect", "x": 0, "y": 0, "width": 40, "height": 20 },
    { "id": 3, "type": "polyline", "points": [[0,0],[20,0],[20,15]], "closed": false },

    { "id": 5, "type": "circle", "cx": 50, "cy": 30, "r": 12 },
    { "id": 6, "type": "arc", "cx": 50, "cy": 30, "r": 12, "startAngle": 0, "endAngle": 90 },

- 読み込み時、欠けている設定は既定値で補完し、`format`/`version` 不一致は明確なエラーメッセージで拒否する。

## 16. 確定した実装詳細(旧TODO の決着)

- 円=中心+半径点、円弧=中心+始点+終角の各クリック指定(3点指定等は未対応)
- Undo履歴上限: 100件 / グリッド自動切替閾値: 8px / 寸法矢印: 塗り三角・長さ3mm(用紙mm)
- フォント: Yu Gothic UI / Meiryo(システムフォント参照、埋め込みなし)
- 楕円: 軸平行(cx,cy,rx,ry)。回転楕円は未対応 / スプライン: Catmull-Rom補間(制御点を通る)
- 部品表: 作成時に既存バルーンの番号から行を自動生成(以後の自動同期はなし)。バルーンは自動採番
- ハッチング: 角度・間隔を作成時に指定。境界は作成時のコピー(元図形の変更に追従しない)
- 印刷余白: `@page margin 0`・用紙実寸指定。倍率100%指定は利用者の操作に依存
- 描画パフォーマンス: 全再描画方式。実用規模(数百要素)では問題なし

## 17. 既知の制約と今後の候補

- 寸法・ハッチングは作成時の形状コピーを保持し、元図形の変更に**追従しない**
- 表題欄の項目・配置カスタムUI、任意レイヤーの追加、回転楕円、任意角度回転、溶接記号は未対応
- 上書き保存はChrome/Edgeのみ(他ブラウザはダウンロード)。ファイルのダブルクリック起動は不可(D&Dで代用)
