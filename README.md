# 製図ツール (seizu-tool)

Excel感覚でグリッドに線を引ける軽量2D製図ツール。単一HTMLで動作。
要件: [drawing-tool-requirements.md](drawing-tool-requirements.md)

## 使う

`dist/seizu.html` をブラウザ(Chrome/Edge推奨)で開くだけ。配布もこのファイル1つをコピーするだけ。

### 基本操作

| 操作 | 方法 |
|---|---|
| 作図 | 直線/連続線/矩形/円/円弧/楕円/**楕円弧**(中心→コーナー→開始→終了)/スプライン/文字 をツールバーから |
| 数値作図 | 下部パネルに始点X/Y・長さ・角度を入れて「作図」(作図中はEnterで数値確定) |
| 数値編集 | 図形を1つ選択するとパネルに値が入る。値を変えて**Enter**か欄を離れると即反映。線=始点+長さ+角度 / 円=中心+**直径** / 円弧=中心+半径+開始角/終了角 / 矩形=左下+幅+高さ+**回転** / 楕円・楕円弧=中心+半径XY+**回転**(+開始/終了角) / 文字=位置+**回転** |
| 線分編集 | 選択中の連続線/スプラインを**もう一度クリック**すると線分単位で選択(オレンジ表示)され、その線分の始点X/Y・長さ・角度を個別に数値指定できる |
| コピー&ペースト | Ctrl+Cでコピー、Ctrl+Vでマウス位置に貼り付け。図形を**右ドラッグ**して離した位置に複製も可 |
| 線種 | 線種セレクタ(外形線/かくれ線/中心線/想像線/補助線)。補助線は印刷されない |
| 移動(パン) | スクロールで上下(チルト/トラックパッドで左右)、矢印キー、中ボタン(Space)+ドラッグ |
| 拡大縮小 | Shift+スクロール または Ctrl+スクロール(カーソル位置基準)、＋/－ボタン、「全体」で用紙全体表示。グリッドは 10/5/2/1/0.5/0.1mm 自動切替 |
| 選択 | クリック / 空白からドラッグで範囲選択(Shiftで追加) |
| 編集 | ドラッグ移動・Ctrl+D複製・Delete削除・回転90°・左右/上下反転 |
| トリム/延長/オフセット | 対象は**直線**(矩形・連続線は先に「分解」)。オフセットは距離を入れて対象クリック→**ずらす側をクリック**の2段階 |
| フィレット/面取り | 半径(サイズ)を入力して**2本の直線を順にクリック**(角がR/面取りになる)。矩形の角は先に「分解」 |
| 分解 | 選択した矩形・連続線をバラバラの直線にする(トリム/フィレット等の前処理) |
| ハッチング | 角度・間隔を指定して閉じた図形(矩形/円/楕円/閉じた連続線)をクリック |
| バルーン/部品表 | バルーンは対象→円位置の2クリック(自動採番)。部品表はクリック配置しセルをダブルクリックで記入 |
| 記号 | 粗さ(Ra記号)/公差枠(ダブルクリックで `//|0.05|A` 形式で編集)。溶接記号は引出線+テキストで |
| スナップ | グリッド/点スナップ(端点・中点・中心・四半点・交点) |
| 中心線モード | 線種「中心線」で作図中は、近くの辺の中点・円の中心にマーカーと十字軸が出て、**軸上ならどこでも吸着**(図形からはみ出して中心線を引ける)。端点への通常スナップは無効化 |
| 投影ガイド | 図形を選択すると特徴点から縦横の補助線。45°線配置で奥行き転写 |
| 寸法 | 寸法(2点+配置点、Shiftで平行寸法)/φ/R/角度(頂点+2点+半径)/C/引出線。値はダブルクリックで上書き(公差は `50±0.1` のように記入) |
| 表題欄 | 欄をダブルクリックで記入(尺度・用紙は自動反映) |
| レイヤー | 「レイヤー」から表示/印刷を切替 |
| Undo / Redo | Ctrl+Z / Ctrl+Y |
| 保存 / 開く | Ctrl+S / Ctrl+O(Chrome/Edgeは同じファイルへ上書き)。D&Dでも開ける |
| クラッシュ復元 | 起動時に未保存の変更があればバナーから復元 |
| 出力 | 「SVG出力」/「印刷」(印刷は倍率100%・実際のサイズを指定すると実寸) |

## 開発

- `npm install` — 初回のみ
- `npm test` — ロジックのテスト
- `npm run dev` — http://localhost:8000 で開発サーバー
- `npm run build` — `dist/seizu.html` を生成
